# robsoriano.com

My personal website.

## Installing

1. Install dependencies

   ```bash
   npm install
   ```

2. Fire up the server and watch files

   ```bash
   npm run dev
   ```

## Built with

- [React](https://reactjs.org/)
- [Next.js](https://nextjs.org/)
- [GitHub GraphQL API](https://developer.github.com/v4/)
- [Tailwind](https://tailwindcss.com/)
